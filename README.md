# Code Review Tune

A dataset and fine-tuning project for training language models on code review tasks.

## Overview

This project contains resources for fine-tuning language models to perform code reviews, generate review comments, and provide constructive feedback on code changes.

## Dataset

The `dataset/` directory contains training data for code review tasks.

## Getting Started

```bash
# Clone the repository
git clone https://github.com/Pratham-26/code_review_tune.git
cd code_review_tune
```
